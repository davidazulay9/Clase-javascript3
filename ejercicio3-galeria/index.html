Instrucciones para el estudiante

APIs a integrar

  • Unsplash API: https://api.unsplash.com/
  • Pixabay API: https://pixabay.com/api/
  • Registra tu API key gratuita en cualquiera de estos servicios

Funciones a implementar

  • fetchImages(query, page) - Obtener imágenes de la API
  • renderImages(images) - Mostrar imágenes en la galería
  • searchImages() - Buscar por término
  • changePage(page) - Navegación de páginas
  • openModal(imageData) - Mostrar imagen ampliada

Características bonus

  • Lazy loading de imágenes
  • Favoritos en localStorage
  • Filtros avanzados
  • Descarga de imágenes
  • Modo infinito scroll

Conceptos clave

  • Async/Await y Promises
  • Fetch API y manejo de errores
  • DOM manipulation avanzada
  • Event delegation
  • Intersection Observer (lazy loading)
