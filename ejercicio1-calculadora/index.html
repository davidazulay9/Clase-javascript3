Calculadora Interactiva

Instrucciones para el estudiante:

  • Completa las funciones en el archivo script.js
  • Implementa la función appendToResult() para agregar números y operadores
  • Implementa la función calculateResult() para realizar los cálculos
  • Implementa la función clearResult() para limpiar la pantalla
  • Implementa la función deleteLast() para borrar el último carácter
  • Bonus: Agrega validación para evitar errores de sintaxis
